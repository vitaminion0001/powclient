# powclient

## Install
```
Install dependancy's first:
sudo apt install golang git

Clone the repository:
git clone https://github.com/vitaminion0001/powclient.git

Go into the directory:
cd powclient

Start building:
go get github.com/vitaminion0001/powclient
go build main/main.go 
nohup ./powclient -server=127.0.0.1:7076 -type=gpu &
```
